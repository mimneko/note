ソースコード

書き方

```
#include <stdio.h>

int main(void){
    printf("hello world!\n");
    return 0;
}
```

```c
#include <stdio.h>

int main(void){
    printf("hello world!\n");
    return 0;
}
```

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello,World!);
    }
}
```

表示

#include <stdio.h>

int main(void){
    printf("hello world!\n");
    return 0;
}

#include <stdio.h>

int main(void){
    printf("hello world!\n");
    return 0;
}

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello,World!);
    }
}
